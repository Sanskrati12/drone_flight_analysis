# 🚁 Drone Flight Analysis using Machine Learning

## Overview
This project builds a drone monitoring system using IMU sensor data.

It focuses on:
- Stability classification (safe flight)
- Battery consumption modeling (efficient flight)

## Features
- Feature engineering using accelerometer and gyroscope data
- Stability prediction using classification models
- Battery consumption prediction using regression
- Remaining flight time estimation

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Key Results
- Accurate stability classification
- Realistic battery depletion simulation
- Dynamic estimation of remaining flight time

## Future Improvements
- Use real battery labels from PX4 logs
- Incorporate time-series models
- Deploy real-time monitoring system
